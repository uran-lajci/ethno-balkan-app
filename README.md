## Setup and Installation

---

This repository requires `python>=3.10`. To prevent dependency conflicts with other packages, It is recommended to
create a virtual environment for this package:

```shell
conda create -n ethno_balkan python=3.10
```

The minimum requirements can be installed with the following command:

```shell
cd ethno-balkan-app/
pip install -e .
```
